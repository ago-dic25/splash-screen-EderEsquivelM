# blank-template

Instrucciones:

Localiza app.json

Crea tu propio logo o imagen (recomendado 1024x1024px PNG) - Puedes utilizar Figma.com o Canva para crear una imagen o modificar algna que ya exista

Guárdala dentro de la carpeta /assets/ y nómbrala splash.png.

Expo debería mostrar la imagen

Utiliza el código en este repositorio para agregar el Splash screen y modificar el tiempo de reproducción

Retos:

Cambia el color de fondo y la propiedad de la imagen entre "cover" y "contain"
Utiliza expo-linear-gradient si quieres utilizar gradientes. Puedes utilizar
